\doxysubsection{Detailed Description}
This is a simple thread-\/safe tool for counting time spent in various chunks of code. This is different from external profiling tools in that it allows the user to count time spent in various bits of code (sub-\/function granularity) or count how many times certain pieces of code are executed. 

This instance will call \mbox{\hyperlink{classfcl_1_1detail_1_1Profiler_acb16ca5075417f33380d4b7cccf13c6f}{Profiler\+::start()}} when constructed and \mbox{\hyperlink{classfcl_1_1detail_1_1Profiler_a98d9edf4ee5b3725420b97fb384acb15}{Profiler\+::stop()}} when it goes out of scope. If the profiler was already started, this block\textquotesingle{}s constructor and destructor take no action.

This instance will call \mbox{\hyperlink{classfcl_1_1detail_1_1Profiler_a1047e6e2d8bb89ea866d2a82a5016a48}{Profiler\+::begin()}} when constructed and \mbox{\hyperlink{classfcl_1_1detail_1_1Profiler_ae74c777eca7092b12169121e52f31710}{Profiler\+::end()}} when it goes out of scope.
