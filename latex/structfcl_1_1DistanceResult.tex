S \mbox{\hyperlink{structfcl_1_1DistanceResult}{fcl\+::\+Distance\+Result}}$<$ S $>$\+::min\+\_\+distance}



Minimum distance between two objects. 

The distance is correct as a positive value when the two objects are not in collision. If they are in collision, min\+\_\+distance would be a implementation defined negative value, which might not the correct negative distance. In order to get correct negative distance set \mbox{\hyperlink{structfcl_1_1DistanceRequest_a2771f4703621c10e1513605289821087}{Distance\+Request\+::enable\+\_\+signed\+\_\+distance}} to true.
