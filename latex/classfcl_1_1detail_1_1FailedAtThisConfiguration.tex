\doxysubsection{Detailed Description}
A custom exception type that can be thrown in low-\/level code indicating that the program has reached an unexpected configuration and that the caller that ultimately has access to the collision objects and their transforms should report the configuration that led to the exception.

This is strictly an {\itshape internal} exception; it should {\itshape always} be caught and transformed into an exception that propagates to the operating system.

Recommended usage is to throw by invoking the macro F\+C\+L\+\_\+\+T\+H\+R\+O\+W\+\_\+\+U\+N\+E\+X\+P\+E\+C\+T\+E\+D\+\_\+\+C\+O\+N\+F\+I\+G\+U\+R\+A\+T\+I\+O\+N\+\_\+\+E\+X\+C\+E\+P\+T\+I\+ON defined below. Code that exercises functionality that throws this type of exception should catch it and transform it to a {\ttfamily std\+::logic\+\_\+error} by invoking Throw\+Detailed\+Configuration(). ~\newline
