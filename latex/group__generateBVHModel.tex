\doxysubsection{Detailed Description}
Create a \mbox{\hyperlink{classfcl_1_1BVHModel}{B\+V\+H\+Model}} using geometric primitives. 

The functions in this group can be used to add geometric primitives (\mbox{\hyperlink{classfcl_1_1Box}{Box}}, \mbox{\hyperlink{classfcl_1_1Sphere}{Sphere}}, \mbox{\hyperlink{classfcl_1_1Ellipsoid}{Ellipsoid}}, \mbox{\hyperlink{classfcl_1_1Cylinder}{Cylinder}}, \mbox{\hyperlink{classfcl_1_1Cone}{Cone}}) to a \mbox{\hyperlink{classfcl_1_1BVHModel}{B\+V\+H\+Model}}. It can either close off the model or leave it unfinalized in order to add more primitives later. \begin{DoxyNote}{Note}
All functions in this group have a common sub-\/set of parameters (listed below). In addition, each has unique parameters related to the geometry type being added and how it should be tessellated. These additional parameters are documented with their corresponding function 
\end{DoxyNote}
\begin{DoxyWarning}{Warning}
If this function is used to create a \mbox{\hyperlink{classfcl_1_1BVHModel}{B\+V\+H\+Model}} containing multiple geometric primitives, the \mbox{\hyperlink{classfcl_1_1BVHModel}{B\+V\+H\+Model}} inherently represents the {\itshape union} of those primitives. The \mbox{\hyperlink{classfcl_1_1BVHModel}{B\+V\+H\+Model}} structure does not retain any notion of the original geometric primitive. 
\end{DoxyWarning}
